# codingtest-study

### 🙋‍♀️ About study

- 책: 이것이 코딩테스트다
- 목적 : 코딩테스트에 자주 나오는 유형 5주안에 한번 돌리기
- 규칙
  - 한 유형당 4시간씩 투자
  - 각 유형에 해당하는 대표적인 풀이법은 외워버리기
  - 디스코드로 만나서 각자 풀이한 방식 설명 + 외운 풀이 테스트보기
  - 모르는 개념 있으면 정리해서 서로 설명해주기
  - 지치지 않고 꾸준하게 하기
